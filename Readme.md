# 🏠 Airbnb Clone (SwiftUI)

Airbnb의 주요 화면을 SwiftUI로 클론하여 UI 구성 능력 연습 
기능 구현 없이 **화면 구성(UI)** 중심으로 제작하였습니다.

---

## ✨ 주요 화면
- 홈 화면 (숙소 리스트)
- 숙소 상세 화면
- 이미지 컴포넌트
- 상단 필터 및 카테고리 탭

---

## 🛠️ 사용 기술 (Tech Stack)
- **Swift**
- **SwiftUI**
- MVVM 구조 기반의 화면 구성
